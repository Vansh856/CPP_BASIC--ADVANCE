The Concept:
Header Files: We've been using #include <iostream>. This simply tells the compiler to copy the contents of the iostream file into our program before compiling. There are two main types:
System header files: These come pre-packaged with the compiler (like <iostream>).
User-defined header files: Files that you will eventually write yourself (included using quotes, like "myHeader.h").
Operators: These are symbols that tell the compiler to perform specific mathematical or logical manipulations.
Arithmetic Operators: + (Add), - (Subtract), * (Multiply), / (Divide), % (Modulo - gives the remainder).
Assignment Operators: = (Used to assign a value to a variable).
Comparison Operators: == (Is equal to), != (Is not equal to), >, <. These return a boolean (1 for true, 0 for false).
Logical Operators: && (Logical AND), || (Logical OR).
