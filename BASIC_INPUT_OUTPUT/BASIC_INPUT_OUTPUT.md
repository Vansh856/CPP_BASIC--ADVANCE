The Concept: In C++, we use "streams" of data to handle input and output.
Output (cout): You already know std::cout (Character Output). It is used to send data to the standard output stream (usually your screen). It is used together with the insertion operator <<.
Input (cin): To get data from the user (usually via the keyboard), we use std::cin (Character Input). It is used together with the extraction operator >>.
When your program reaches a cin statement, it will pause and wait for the user to type something and press Enter before moving on to the next line of code.
