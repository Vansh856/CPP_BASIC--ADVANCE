The Concept:
Data Types: In our last lesson, we stored your age using int, which stands for "integer" (a whole number). C++ is strict about what kind of data a variable holds. If you want to store a decimal number, a single letter, or a true/false value, you need to use different data types
.
Variable Scope: "Scope" simply refers to the region in your code where a variable is accessible
.
Local Scope: If you create a variable inside a function (like inside the {} of your main() function), it is a local variable. It can only be used inside those specific brackets.
Global Scope: If you create a variable outside of any function (above int main()), it is a global variable. Any function in your entire program can access it.
