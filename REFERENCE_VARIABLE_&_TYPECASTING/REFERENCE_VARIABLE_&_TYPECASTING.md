The Concept:
Reference Variables: Imagine you have a nickname. Whether people call you "Vansh" or by your nickname, they are still talking to the exact same person. A reference variable is exactly that—an alias or an alternate name for an already existing variable
. If you change the value using the reference, the original variable's value changes too, because they share the exact same memory location!
Typecasting: This is the process of converting a variable from one data type to another
. For example, if you have a float (like 4.56) but you only want to use the whole number part (4), you can "cast" it into an int.
